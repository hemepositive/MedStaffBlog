**Intended Audience: All Clinical Staff**

When accessing "Outside Records" it will now link to CommonWell. CommonWell is a national record locator service, essentially it connects specific patient data from other Electronic Medical Records (EMRs) into the PowerChart system. CommonWell pulls from EMR sources across the country. This record information will be added to a new "Summary" tab on the "Outside Records" menu section. Records can be viewed and/or saved to the chart. Please note that information pulled from CommonWell is **NOT** a full medical record from another facility, but a summary that may include allergies, medications, procedures, immunizations, provider documentation, vitals, etc. Follow the steps below to learn how to access the CommonWell data.

## Accessing Outside Records

1. Select **"Outside Records"** from the blue menu bar.

![Outside Records menu item](/images/outside-records/page1_img1.png)

This will open the Outside Records M-Page

![Outside Records M-Page](/images/outside-records/page1_MPage.png)

2. The patient's CommonWell enrollment status will appear in the top right of the page.

![CommonWell Enrolled status](/images/outside-records/page1_img2.png)

3. Select the **"Refresh"** icon to query for any new documents from CommonWell. **"CommonWell: New Sources"** will display if the patient has any additional sources/facilities that have not been linked to the patient. Select this link to add those sources.

![CommonWell New Sources link](/images/outside-records/page1_img3.png)

## Confirming Sources

1. In the **"Confirmed by:"** drop down, select if the patient or another individual confirmed the patient has been to the facility listed.
2. Select the **"Confirm"** check box next to the source.
3. Select **"Update Sources"** to add that facility.

![Confirm sources and Update Sources](/images/outside-records/page1_img4.jpeg)

4. The **"View Confirmed Sources"** page will open, and the document will be added in alphabetical order. Select the **"Close"** button when done.

![View Confirmed Sources list](/images/outside-records/page2_img1.jpeg)

## Enrolling a Patient in CommonWell

If it indicates that the patient is not enrolled in CommonWell, you can easily perform a manual enrollment from this screen:

1. Select the **"CommonWell: Not Enabled"** link.

![CommonWell Not Enabled link](/images/outside-records/page2_img2.png)

2. Verify that the patient Full Name and information from PowerChart matches the patient shown in the search.
3. If this matches, choose **"Link This Record"** link on the left side, and then click **"Close."**
4. If there is a confirmed source found for this patient, it will then display it for you to see.

![Link This Record screen](/images/outside-records/page2_img3.png)

5. After enrolling the patient in CommonWell, click the button to cycle the query again.

![Last query refresh button](/images/outside-records/page3_img1.png)

## Viewing Documents

1. To open a specific document, select the **"Summarization of episode note"** hyperlink.

![Document list with Summarization of episode note links](/images/outside-records/page3_img3.png)

2. Use the links in the grey menu to jump to specific parts of the **"Continuity of Care Document"** (CCD).
3. The document can be printed, if desired, by selecting **"Print"**.
4. To return to the "Outside Records" page, select the **"x"** at the top of the window.

![Continuity of Care Document view](/images/outside-records/page3_img2.jpeg)

## Using the Summary View

1. Select **"Summary"** to view data from the CCDs separated by different categories (this is the easiest way to see aggregated patient care information).

![Summary view with aggregated patient data](/images/outside-records/page4_img3.jpeg)

2. Use the drop down filters to narrow or expand the data that displays on the Summary.

![Summary filter dropdowns](/images/outside-records/page4_img1.png)

3. To view more information, any data that displays in **blue** can be selected to view the data in a separate window.

![Social History detail view example](/images/outside-records/page4_img2.png)

## Reconciling Data into PowerChart

To reconcile the data into the patient's record in PowerChart:

1. Select **"Reconciliation"**.

> **Note:** If you view data from CommonWell, please reconcile the data into PowerChart. Any **"purple diamond"** means that there is data that needs to be reconciled.

![Reconciliation link](/images/outside-records/page5_img2.png)

2. Once open, select from the **"Unverified Data from Outside Sources"**.

![Reconciliation View with Unverified Data from Outside Sources](/images/outside-records/page5_img3.jpeg)

3. Once selected, the details will open. Choose one of the following:
   - **"Add"** to add the data to the patient's record
   - Use the dropdown to select **"Add with Changes"**
   - **"Discard"** if the data does not need to be added

![Add, Add with Changes, or Discard options](/images/outside-records/page6_img1.jpeg)

4. Once all options are either added or discarded, the "Purple Diamond" will disappear. Continue to additional sections to reconcile other data into the patient's chart.

> **IMPORTANT:** Select the **"Complete Reconciliation"** button below that section for the changes to take effect. Even if every item is not accepted or discarded, a reconciliation is completed by ensuring this button is selected.

![Complete Reconciliation button](/images/outside-records/page6_img2.png)
